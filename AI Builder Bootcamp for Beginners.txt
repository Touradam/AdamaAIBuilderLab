AI Builder Bootcamp for Beginners

### From “Never Used AI” → “Building & Deploying Real Apps”

Your idea is excellent because most people are overwhelmed by AI and coding. The best approach is not teaching programming theory first — it is teaching them how to *build useful things quickly using AI tools*.

You can position this as:

> “Learn how to use AI to build websites, apps, and automation tools without needing a computer science background.”

# Suggested Program Structure

# Phase 1 — AI Productivity Foundations (Week 1)

## Objective

Help students become comfortable using AI tools.

## Topics

* What AI can and cannot do
* Prompt engineering basics
* How developers use AI
* Using ChatGPT effectively
* Using Cursor
* Understanding files/folders/projects
* Basic terminal usage

## Mini Projects

* AI-generated resume website
* AI-generated personal bio page

## Outcome

Students realize:

> “I can already build things.”

---

# Phase 2 — Build Your First Website (Week 2)

## Objective

Teach web fundamentals through AI-assisted development.

## Concepts

* HTML basics
* CSS basics
* JavaScript basics
* Website structure
* Responsive design

## Cursor Workflow

Teach students:

* How to prompt Cursor
* How to ask for edits
* How to debug errors
* How to improve design

## Projects

### Beginner

* Portfolio website
* Startup landing page

### Intermediate

* Solar startup homepage
* Restaurant website
* Personal brand site

## Outcome

Students deploy a real website online.

---

# Phase 3 — Git & GitHub (Week 3)

## Objective

Teach professional workflow basics.

## Topics

* What is version control?
* Git basics
* Commits
* Branches
* Push/pull
* GitHub repositories

## Practical Workflow

Students learn:

1. Build locally
2. Save changes with Git
3. Push to GitHub
4. Collaborate professionally

## Mini Project

* Publish their portfolio repository

---

# Phase 4 — Deploy Websites Online (Week 4)

## Objective

Teach students how to launch projects publicly.

## Topics

* Domains
* Hosting
* Deployment
* Environment variables

## Platforms

* [Hostinger](https://www.hostinger.com?utm_source=chatgpt.com)
* GitHub Pages
* Optional:

  * [Vercel](https://vercel.com?utm_source=chatgpt.com)
  * [Netlify](https://www.netlify.com?utm_source=chatgpt.com)

## Projects

Students deploy:

* Their portfolio
* Startup landing page

## Outcome

Students have:

* A public website
* A portfolio
* Confidence

# Teaching Strategy

## DO NOT Start With Heavy Theory

Beginners quit when overwhelmed.

Instead:

* Build first
* Explain concepts gradually
* Focus on outcomes

---

# Recommended Learning Flow

For every lesson:

## 1. Show

Demonstrate the final product.

## 2. Build Together

Students follow along.

## 3. Explain

Teach concepts during building.

## 4. Improve

Students customize independently.

---

# Important Mindset You Should Teach

## AI Does NOT Replace Thinking

Teach students:

* AI accelerates development
* Humans provide direction
* Prompt quality matters
* Debugging is still important
