Get Ready

Before our first session, here's what you'll need, what to expect, and what you'll walk away with after one month.

Requirements

What You Need to Succeed

This program works best when you show up prepared. None of this is meant to scare you off — it's here so you know exactly what you're signing up for.

  • A personal computer — this is not a phone-only program
  • Ideally two screens: one to watch, one to practice
  • Stable internet connection for live sessions
  • Willingness to use ChatGPT and the OpenAI API
  • A Cursor account (we'll set this up together if needed)
  • Budget for $20 for one month of Cursor
  • Commitment of 5 hours per week — sessions plus practice
  • Be fully present during sessions — cameras on, distractions off
Expectations

This Is Not a Class to Watch

If you're looking for something to play in the background while you do other things, this isn't it.

This is hands-on from the first minute. You'll build real projects — not toy examples — and ship things you can point to and say, "I made that."

I'll walk you step by step, hand in hand, toward becoming someone who can build with AI — not someone who watches others build.

Come ready to type, experiment, ask questions, and push through the moments when it feels hard. That's where the growth happens.

Outcomes

By the End of the 1-Month Program, You Will Be Able To…

  • →Build and deploy a simple website
  • →Build a basic web app that uses an AI API
  • →Understand the basics of prompting and model usage inside an application
  • →Use Git and GitHub confidently — commit, push, branches, and pull request basics
  • →Deploy via GitHub Pages and understand alternative hosting options
  • →Build a simple Chrome extension prototype
  • →Understand fundamentals of how AI and ML work at a high level — training vs inference, limitations, and evaluation
  • →Work effectively with Cursor as an AI coding partner
